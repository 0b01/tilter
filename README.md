# AirPods Pro Motion Sample

AirPods Pro のモーションセンサの値を取得するサンプルプロジェクト

## Requirements 必要事項
 - Xcode: 12.0
 - iOS  : 14
 - 機材
 - iPhone, iPad
 - AirPods Pro
  
## Reference
 - [Apple Developer Documentation](https://developer.apple.com/documentation/coremotion/cmheadphonemotionmanager)
